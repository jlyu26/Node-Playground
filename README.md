# Node-Playground

Side projects implemented while playing build-in and NPM modules with Node.js .

## notes-alrd

A command-line notes system, support 'add', 'list', 'read', 'delete' operations.

- Lodash ([ _ ](https://lodash.com/docs/4.17.5))
